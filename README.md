# Dynamic E-Commerce Web Application

## Overview
This project is a dynamic e-commerce web application built with React, Redux, and React Router. It allows users to browse through a wide range of products, manage their cart, mark products as favorites, and enjoy a responsive user interface.

## Key Features
- **Product Catalog:** Explore a vast collection of products fetched from an external API.
- **Cart Management:** Add products to the cart, update quantities, and view a summary of items.
- **Favorites:** Mark products as favorites and access them easily in the favorites section.
- **Responsive Design:** Ensures a sleek and responsive user interface across devices.

## Tech Stack
- **React:** Building reusable UI components and managing state.
- **Redux:** Global state management, especially for the cart functionality.
- **React Router:** Client-side routing for seamless navigation between pages.
- **Context API:** Managing product-related state, such as fetching products and handling favorites.
- **Tailwind CSS:** Provides utility classes for styling, ensuring a clean and modern design.

## Impressions
- Explore a vast collection of products with a user-friendly interface.
- Seamlessly manage your cart and keep track of your favorites.
- Experience a smooth and responsive browsing experience across devices.
